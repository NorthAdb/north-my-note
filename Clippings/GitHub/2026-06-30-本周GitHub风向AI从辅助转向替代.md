## 1️⃣ deer-flow ⭐76k — 字节跳动的 SuperAgent

[项目地址](https://github.com/bytedance/deer-flow)

字节跳动开源的长周期 SuperAgent 框架。用 LLM graph 驱动，支持子 agent 沙箱执行，可接入飞书、微信。核心能力是将复杂任务自动分解给子 agent 并行执行，最后汇总结果。

对于学生来说，读透这份源码，系统设计面试能聊的东西直接上升一个维度。

---

## 2️⃣ Penpot ⭐55k — 开源设计工具

[项目地址](https://github.com/penpot/penpot)

对标 Figma，纯开源，浏览器直接跑或自托管。用 Clojure 编写，支持 SVG/CSS/HTML 设计、实时协作、MCP 服务器打通设计到代码。

学生团队做项目时，设计师做好界面，开发者直接拿代码走，不再有设计稿和代码脱节的问题。

---

## 3️⃣ daily_stock_analysis ⭐53k — 零成本股票分析

[项目地址](https://github.com/ZhuLinsen/daily_stock_analysis)

LLM 驱动，支持 A 股/港股/美股/日股/韩股，聚合行情、新闻、技术指标，自动生成买卖决策看板并推送到企业微信/飞书。用 GitHub Actions 定时运行，零成本。

示范了一个完全自动化、零运维成本的 AI 系统的完整架构，做毕设可以直接参考。

---

## 4️⃣ Agent-Reach ⭐50k — AI 全网信息采集

[项目地址](https://github.com/Panniantong/Agent-Reach)

给 AI agent 装上"眼睛和耳朵"。能读 Twitter、Reddit、YouTube、小红书、GitHub，全部免费不需要 API key。平台被封自动切换备用方案，完全无感知。

写调研报告时让 agent 自己跑一遍，回来给你一份整理好的摘要，信息获取成本趋近于零。

---

## 5️⃣ Voicebox ⭐37k — 开源语音工作室

[项目地址](https://github.com/jamiepine/voicebox)

7 种 TTS 引擎、23 种语言、零样本声音克隆。Tauri + Rust 构建，本地运行不联网。支持 pitch、reverb、delay 等效果。

给 AI agent 装上声音，或给项目加语音交互模块，直接拿来用，不用把数据交给第三方。

---

## 6️⃣ OpenMontage ⭐32k — 智能视频制作

[项目地址](https://github.com/calesthio/OpenMontage)

用自然语言描述需求，自动完成从策划、分镜、素材剪辑到最终渲染的全流程。12 条生产流水线、52 个工具、500+ agent 技能。

"做视频"从专业技能变成了聊天技能，本周一周涨了 18,000 ⭐。

---

## 7️⃣ cognee ⭐26k — AI 长期记忆平台

[项目地址](https://github.com/topoteretes/cognee)

AI agent 的长期记忆平台，跨会话记住重要信息，支持知识图谱构建和语义检索。在 BEam 基准测试中得分 0.79（100K token 场景），干掉了之前 SOTA。

解决了 agent 最大的痛点——每次对话都是一张白纸。架构设计值得读一遍源码。

---

## 8️⃣ codebase-memory-mcp ⭐25k — 代码知识图谱

[项目地址](https://github.com/DeusData/codebase-memory-mcp)

MCP 服务器，把你的代码库索引成知识图谱。支持 158 种语言，查询响应不到 1 毫秒，token 消耗降低 99%。Linux 内核 2800 万行代码，3 分钟索引完。

不是帮你写代码，是帮你理解代码。读开源项目源码的时间从几周变成几小时。

---

## 9️⃣ ai-website-cloner-template ⭐25k — 一键克隆网站

[项目地址](https://github.com/JCodesMore/ai-website-cloner-template)

TypeScript + Next.js 16 + Tailwind CSS v4。一行命令克隆目标网站，AI 自动分析设计规范，并行构建每个区块，合并成干净的代码。

前端开发的速度正在被 AI 拉到秒级。
