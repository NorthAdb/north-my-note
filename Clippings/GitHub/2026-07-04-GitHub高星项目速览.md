## 🔮 2026 年趋势观察

### 1. AI Agent Skills 生态爆发
2026 年最大的现象是 AI 编码 Agent（Claude Code / Codex CLI / Cursor / OpenClaw）的 **skills/plugin 生态** 爆发式增长。多个 skills 合集项目获得 50k+ 甚至 200k+ 的 stars。这类似于 2023 年 ChatGPT Plugin 的热潮，但深度更深——Skills 直接嵌入开发工作流。

### 2. Token 优化成为刚需
Claude Code / Codex 等 agent 的使用导致 API token 消耗巨大，催生了 [caveman](https://github.com/JuliusBrussee/caveman)（83k⭐）、[rtk](https://github.com/rtk-ai/rtk)（68k⭐）、[headroom](https://github.com/headroomlabs-ai/headroom)（56k⭐）等 token 压缩/优化工具。

### 3. 代码知识图谱 + MCP
[codebase-memory-mcp](https://github.com/DeusData/codebase-memory-mcp)（周增 10k⭐）等 MCP 服务器项目火爆，说明 Agent 需要 **持久化、结构化的代码理解**，而非简单的上下文窗口。

### 4. 多 Agent 协作
从 [orca](https://github.com/stablyai/orca)、[oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode) 等项目可见，**多 Agent 并行协作**成为主流范式——不再是一个 Agent 单干，而是多个专业 Agent 组成团队。

### 5. DESIGN.md：设计系统新规范
Google 推出的 [design.md](https://github.com/google-labs-code/design.md) 规范（24.7k⭐）标志着 **向 AI Agent 描述设计系统** 的需求正式标准化，类似的 [open-design](https://github.com/nexu-io/open-design)（74.8k⭐）则试图替代 Claude Design。

### 6. 安全与 Agent 结合
[strix](https://github.com/usestrix/strix)（34.9k⭐，周增 7.6k）代表了 AI Agent 在安全测试领域的应用趋势——让 Agent 主动发现漏洞。
