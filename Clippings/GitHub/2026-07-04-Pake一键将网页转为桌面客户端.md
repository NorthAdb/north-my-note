## 这是什么？

Pake 可以把任何网页打包成一个**桌面客户端应用**。一条命令，网页变 App。

```bash
# 基本用法
pnpm install -g pake-cli
pake https://github.com --name GitHub

# 高级用法（自定义图标、窗口尺寸等）
pake https://weekly.tw93.fun --name Weekly \
  --icon icon.icns --width 1200 --height 800 --hide-title-bar
```

用 Rust + Tauri 构建，生成的客户端体积小、性能好，比 Electron 方案轻量得多。
