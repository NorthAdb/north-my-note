# 📘 north-my-note

个人 Obsidian 知识库 —— 基于**双向链接**的笔记系统。

## 📂 目录结构

```
north-my-note/
├── README.md           ← 本文件，仓库介绍
├── .gitignore          ← Git 忽略规则
├── 日记/               ← 每日记录、流水日志
├── 项目/               ← 按项目归集的笔记
├── 领域/               ← 知识领域（编程、阅读、写作……）
└── 附件/               ← 图片等媒体资源
    └── 图片/
```

## 🧭 使用指引

| 目录 | 用途 | 示例 |
|------|------|------|
| `日记/` | 每日随记、日报、心情 | `2026-07-04.md` |
| `项目/` | 每个项目一个文件或子目录 | `my-app-开发.md` |
| `领域/` | 跨项目的知识沉淀 | `编程.md`、`阅读笔记.md` |
| `附件/` | 图片等静态资源 | 通过 Obsidian 粘贴自动存放 |

## 🔗 双向链接

利用 `[[wikilink]]` 在笔记间建立连接，Obsidian 图谱视图会自动呈现知识网络。

- 日记可链接到项目：`[[项目/my-app-开发]]`
- 领域笔记可引用日记：`[[日记/2026-07-04]]`
- 跨领域链接：`[[编程]]` → `[[阅读笔记]]`
